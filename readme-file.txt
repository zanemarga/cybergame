# 🛡️ Krāpnieku Pilsēta (ScamCity)

**Kiberdrošības izglītības RPG spēle latviešu valodā**

Interaktīva 2D spēle, kas māca atpazīt un izvairīties no izplatītākajiem interneta krāpšanas veidiem.

## 🎮 Kā Spēlēt

1. Atver `index.html` pārlūkā vai apmeklē [tiešsaistes versiju](https://TAVS-LIETOTAJVARDS.github.io/scamcity)
2. Izmanto **bulttaustiņus** vai **WASD**, lai pārvietotos pa pilsētu
3. Tuvojies NPC ar **❗** virs galvas
4. Spied **SPACE** vai **ENTER**, lai runātu
5. Izvēlies pareizo atbildi, lai aizsargātu pilsētu!

## 📚 Izglītojošais Saturs

Spēle ietver **6 reālus krāpšanas scenārijus**:

| NPC | Krāpšanas Veids |
|-----|----------------|
| 👤 Samulsis Pilsonis | E-pasta pikšķerēšana (phishing) |
| 👵 Satraukts Seniore | VID telefona krāpšana |
| 👔 Biroja Darbinieks | CEO / biznesa e-pasta krāpšana |
| 💔 Vientuļā Sirds | Romantiskā krāpšana |
| 😰 Panikā Nonākušais | Tehniskā atbalsta krāpšana |
| 📦 Tiešsaistes Pircējs | Piegādes SMS krāpšana |

## 🛠️ Tehniskie Dati

- **Valoda:** Latviešu
- **Tehnoloģijas:** HTML, CSS, JavaScript (vanilla — bez bibliotēkām)
- **Fails:** Viens `index.html` fails — nav nepieciešama instalācija
- **Pārlūks:** Jebkurš moderns pārlūks (Chrome, Firefox, Edge, Safari)

## 📁 Failu Struktūra

```
scamcity/
└── index.html    ← Viss vienā failā — vienkārši atver!
```
